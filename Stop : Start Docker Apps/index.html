Stop / Start Docker Apps

Stop app in Docker

To stop Docker from running an app just type:


docker stop <name of the app>

Docker informs you when the app has been stopped by displaying it’s name, for postgres it looks like this


mmj$ docker stop postgres
postgres

Remove app from Docker

mmj$ docker rm postgres
postgres

You may always add posgres to Docker` with Install PostgreSQL In Docker.
