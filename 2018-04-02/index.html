Logos Tweak Logos Tweak

Environmental requirements

Make sure of the following points before using the tool:


  • Install the latest theos

sudo git clone --recursive https://github.com/theos/theos.git /opt/theos

  • Install ldid (if installing the os process installs ldid, skip)

brew install ldid

ssh-keygen -t rsa -P '' ssh-copy-id -i /Users/username/.ssh/id_rsa root@ip

Jailbreak plugin Xcode debugging


 Before the Tweak and CaptainHook project does not support Xcode direct debugging, only through the print log to see the effect, the following can be directly through the MonkeyApp to facilitate debugging a lot, but each debugging must re-install the application is actually very troublesome, here A method to debug Tweak and CaptainHook projects directly using Xcode.


Since Xcode uses the default /Developer/usr/bin/debugserver debugging, the application being debugged must be resigned and installed. You can use MonkeyApp to resign and install it.


The specific process is as follows:


  1. Re-install the debugged application.
  1. Create a Tweak or CaptainHook project, write code, specify the bundle id, and set breakpoints.
  1. Click the Xcode menu bar -> Debug -> Attach to Process by PID or Name.
  1. Output process name or pid.
