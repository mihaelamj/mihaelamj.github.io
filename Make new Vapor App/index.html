Make new Vapor App

Once Vapor Toolbox is successfully installed, the next step is creating a new Vapour app.


Type the command in terminal (in you app’s dir):


	vapor new HelloVapor --branch=beta

—branch=beta is here only temporary, so that we may use vapor version 3.

It will be removed once the version 3 is released.
